start_link/1

start_link(Args) -> Result

  • Args = term()
  • Result = {ok, Pid} | ignore | {error, Error}
  • Pid = pid()
  • Error = {already_started, Pid} | term()

Starts the Arya UDP server. Args are ignored. For now server starts in active mode. That means, process will receive all the packets as Erlang messages, without explicit recv/2 using.
