Description

This is the processors' supervisor of Arya server. It looks after an essential workers processing all the incoming data before it sends to the downloader process (or just dropped, if data is malformed).

For further information about processor module see its documentation.

This modue describes callbacks for supervisor starting and terminating as well as a worker adding method.
